Data Analysis and Visualization in Python

A practical introduction to Data Science

This book introduces concepts and skills that can help you tackle real-world data analysis challenges. It covers concepts from probability, statistical inference, linear regression and machine learning and helps you develop skills such as Python programming, data wrangling with polars, data visualization with plotnine, file organization with UNIX/Linux shell, version control with GitHub, and reproducible document preparation with Jupyter.

Note — work in progress
This book is an adaptation to the programming language Python of the original script Data Analysis and Visualization in R and is still under development. Chapters will be added in order. For any content not yet implemented, please refer to the original. Theoretical concepts are identical, only the programming language is changed.

Prerequisites

Basics in probabilities are required. Chapters 13-15 (“Introduction to Statistics with R”, “Probability” and “Random variables”) of the Book “Introduction to Data Science” https://rafalab.github.io/dsbook/ make a good refresher. Make sure all concepts are familiar to you. Check your knowledge by trying the exercises.
